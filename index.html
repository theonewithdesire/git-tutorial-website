What are Git & GitHub?

Git

Git is a distributed version control system that tracks changes in your code, letting you collaborate seamlessly and maintain code history.

GitHub

GitHub is a cloud platform that hosts Git repositories, making it easy to share code, collaborate with others, and showcase your projects.

Why Should You Use Git?

Track Changes

Never lose your work and always know what changed

Collaborate

Work with others without stepping on toes

Branch & Merge

Experiment safely with new features

Backup

Keep your code safe in the cloud

Essential Commands

Initialize Repository

git init

Create a new Git repository in current directory

Clone Repository

git clone

Download a repository from GitHub

Add Changes

git add

Stage all changes for commit

Commit Changes

git commit

Save staged changes with a description

Push Changes

git push

Upload commits to GitHub

Pull Changes

git pull

Download and merge changes from GitHub

Check Status

git status

View the state of your working directory

View History

git log

Show commit history with details

Switch Branch

git checkout

Switch to a different branch
